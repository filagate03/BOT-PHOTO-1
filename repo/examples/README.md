Сложите сюда jpg/png с названиями из `manifest.json`. Каждый объект имеет поля:

```json
[
  {"style": "fashion", "title": "Fashion", "caption": "Студийная съёмка", "file": "fashion.jpg"}
]
```

Файл должен лежать рядом (например `fashion.jpg`). Пока каталог пустой, бот просто сообщит, что примеры появятся позже.
